# 💙 프로젝트 소개

학교에서 우연히 만나게 된 소녀 **아오이 린**.

플레이어의 선택에 따라  
린의 감정과 표정, 그리고 이야기의 결말이 달라집니다.

각 상황에서 서로 다른 대화와 엔딩이 진행됩니다.

---

# 🌸 엔딩 구성

| 엔딩 | 설명 |
|---|---|
| 해피 엔딩 | 서로의 마음이 가까워지는 결말 |
| 노멀 엔딩 | 어색하지만 감정이 남는 결말 |
| 배드 엔딩 | 결국 멀어져버리는 결말 |
